# HTML

**Etiquetas**

Hay dos tipos de etiquetas, las abiertas y las cerradas. Las etiquetas cerradas tienen un etiqueta de apertura, y otra de cierre:

```html
<etiqueta></etiqueta>
```

las abiertas no tienen ninguna que las cierre:

```html
<etiqueta />
```


**Estructura general**

```html
<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
  </body>
</html>
```
